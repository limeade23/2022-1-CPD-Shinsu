# 2022 Spring Creative Product Design Shinsu

- Webpack + Babel + PostCSS(SCSS)
- Vue3 + Vuex + Vue-Router
- ESLint

## 프로젝트 시작 전 패키지 설치

```bash
npm i
```

## 개발 서버 시작

```bash
npm run dev
```

## firebase Congig 설정

root 디텍토리에 `firebaseConfig` 파일 생성 후 아래와 같은 형식으로 내용 작성

```
export default {
  <!-- firebase const config 파일 init  -->
}
```
